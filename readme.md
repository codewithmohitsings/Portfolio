# About This Project
This is a personal portfolio website for Mohit Rath, built with Vite and React while preserving the original portfolio layout, styling, and behavior through the legacy assets stored in the public folder. The site presents a clean single-page experience with sections for Home, About, and Projects, plus a theme switcher, decorative effects, and smooth navigation.

The project combines modern React structure with the existing HTML/CSS/JavaScript portfolio design, so the original visual style remains intact while the app is easier to develop and maintain. It also loads the old jQuery-based plugins at runtime to keep the legacy interactions working inside the React app.

# Features
Home section with intro, role, and social links
About section with bio, skills, experience, and education
Projects section showcasing selected work
Theme color switcher
Scroll-to-top and loader components
Legacy portfolio assets and scripts preserved in public
Tech Stack
React 18
Vite
HTML, CSS, JavaScript
Bootstrap and jQuery-based legacy plugins
